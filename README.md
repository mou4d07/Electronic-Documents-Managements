# Industrial EDM - Electronic Document Management

ASP.NET Core MVC application for viewing and managing large-format technical plans (A3, A2, A1) of buildings, equipment, and installations in an industrial complex.

## Tech Stack
- Backend: ASP.NET Core MVC (.NET 6/7/8)
- Database: SQL Server
- Frontend: Razor Pages, JavaScript (Viewer.js for large plans)
- Deployment: IIS / Azure

## Prerequisites
- .NET SDK (v6 or later)
- SQL Server (2019+)
- Visual Studio 2022 / VS Code
- Git
